Welcome to Novus Flexy

A Modern Angular Admin Dashboard Template

About This Template

Novus Flexy is a clean, professional admin dashboard template customized by Novus Apps for creating modern web applications. This template provides a solid foundation with essential features and a clean architecture that can be extended to meet your specific needs.

Technology Stack

  • Angular 19 - Latest framework with standalone components
  • Angular Material 19 - Comprehensive UI component library
  • TypeScript 5.6 - Type-safe development
  • SCSS - Advanced styling with Sass
  • ESBuild - Lightning-fast application builder
  • Tabler Icons - Beautiful icon library
  • ApexCharts - Modern charting library
  • Responsive Design - Mobile-first approach

Getting Started

This template includes a pre-configured layout with navigation, authentication pages, and a responsive sidebar. You can start building your application by:

  • Adding new routes in app.routes.ts
  • Creating new components using Angular CLI
  • Customizing the theme in src/assets/scss/
  • Extending the navigation in sidebar-data.ts
